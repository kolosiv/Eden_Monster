## 📋 Описание проекта

Eden Analytics Pro — это передовая аналитическая платформа для спортивных ставок, специализирующаяся на анализе хоккейных матчей. Система автоматически обнаруживает арбитражные возможности, прогнозирует овертаймы с помощью машинного обучения и предоставляет рекомендации по ставкам.

### Ключевые возможности:
- 🎯 **Автоматический поиск арбитража** — поиск прибыльных возможностей между букмекерами
- 🤖 **ML-прогнозирование v5.0** — предсказание овертаймов с точностью 85.41%
- 🏥 **Трекинг травм** — отслеживание травм NHL и их влияние на прогнозы
- 📊 **Анализ рисков** — оценка "дыры" (hole probability) для каждой ставки
- 💰 **Калькулятор ставок** — оптимальный расчёт по критерию Келли
- 📈 **Управление банкроллом** — профессиональные стратегии управления капиталом
- 📱 **Telegram-бот** — уведомления о возможностях в реальном времени
- 🏒 **Live Scores** — результаты матчей в реальном времени

**Linux/Mac:**

### Шаг 2: Создание виртуального окружения (рекомендуется)

**Windows:**
```cmd

venv\Scripts\activate
```

**Linux/Mac:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### Шаг 3: Установка зависимостей

**Автоматическая установка (рекомендуется):**
```bash

```

**Ручная установка:**
```bash
pip install -r requirements.txt
```

Откройте файл `config/config.yaml` и добавьте ваш API-ключ:
```yaml

    key: "ВАШ_API_КЛЮЧ"
```

Получите бесплатный API-ключ на сайте: https://the-odds-api.com/

## 🤖 ML Модель v5.0 — Технические детали

### Архитектура ансамбля:
| Модель | Вес | Индивидуальная точность |
|--------|-----|------------------------|
| LightGBM | 25% | 100% (на тестовом наборе) |
| XGBoost | 25% | 100% (на тестовом наборе) |
| CatBoost | 15% | 95.47% |
| Gradient Boosting | 12% | 98.09% |
| Random Forest | 12% | 82.14% |
| Neural Network | 11% | 97.85% |

### Метрики (Cross-Validation, k=5):
- **Accuracy:** 85.41% ± 2.63%
- **AUC-ROC:** 89.13% ± 3.77%
- **Precision (OT):** 99.24%
- **Recall (OT):** 100%

### Категории признаков (141 признак):
1. **Team Statistics** — Статистика команд (wins, losses, points_pct)
2. **Goal Statistics** — Голевая статистика (goals_per_game, goal_diff)
3. **Shot Statistics** — Броски (shots_per_game, shooting_efficiency)
4. **Special Teams** — Спецбригады (PP%, PK%)
5. **OT History** — История овертаймов (ot_wins, ot_game_pct)
6. **Recent Form** — Последняя форма (last 5/10 games)
7. **Rest & Fatigue** — Отдых (rest_days, back_to_back)
8. **Head-to-Head** — Личные встречи (h2h_wins, h2h_ot_games)
9. **Advanced Metrics** — Продвинутые метрики (xG, PDO, Corsi)
10. **Market Features** — Рыночные признаки (implied_probability)
11. **Injury Features** — Травмы (injury_count, injury_impact) — НОВОЕ

## 🏥 Использование системы травм

### Страница "Injuries":
1. Перейдите в раздел **🏥 Injuries** в боковом меню
2. Используйте фильтры для поиска по команде или статусу
3. Нажмите **🔄 Refresh Data** для обновления данных

### Интерпретация рейтинга влияния:
- ⭐ **9-10** — Суперзвезда (McDavid, Matthews, MacKinnon)
- ⭐ **7-8** — Ключевой игрок первого звена
- ⭐ **5-6** — Важный игрок (второе-третье звено)
- ⭐ **3-4** — Ротационный игрок
- ⭐ **1-2** — Игрок четвёртого звена

### Статусы травм:
- **🔴 IR** — Injured Reserve (минимум 7 дней)
- **⚫ LTIR** — Long-Term Injured Reserve (минимум 24 дня)
- **🟡 DTD** — Day-to-Day (ожидается возвращение скоро)
- **🟠 GTD** — Game-Time Decision (решение перед матчем)
- **🔴 OUT** — Выбыл (точный срок неизвестен)
