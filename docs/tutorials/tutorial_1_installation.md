# 🎥 Видео-туториал #1: Установка Eden Analytics Pro

## 🎬 Метаданные

- **Длительность:** 5-7 минут
- **Уровень:** Начинающий
- **Целевая аудитория:** Новые пользователи
- **Формат:** Запись экрана + голос за кадром

---

## 📑 Сценарий

### Введение (0:00 - 0:30)

**[Заставка с логотипом Eden Analytics Pro]**

**Диктор:**
> "Приветствую вас на канале Eden Analytics! В этом видео мы рассмотрим, как установить Eden Analytics Pro на ваш компьютер и подготовить его к работе. Установка займёт всего несколько минут."

**[Скриншот: обзор интерфейса приложения]**

---

### Часть 1: Системные требования (0:30 - 1:00)

**[Скриншот: список требований]**

**Диктор:**
> "Перед началом убедитесь, что у вас есть:
> - Python версии 3.8 или выше
> - Минимум 4 гигабайта оперативной памяти
> - Стабильное интернет-соединение
> 
> Если у вас ещё не установлен Python, перейдите на python.org и скачайте последнюю версию."

**[Действие: открыть python.org]**

---

### Часть 2: Загрузка проекта (1:00 - 2:00)

**[Действие: открыть страницу GitHub]**

**Диктор:**
> "Есть два способа загрузить проект. Первый — через Git, если он у вас установлен..."

**[Действие: открыть терминал, ввести команду]**

```bash
git clone https://github.com/username/eden-analytics-pro.git
cd eden-analytics-pro
```

**Диктор:**
> "Второй способ — скачать ZIP-архив. Нажмите кнопку Code, затем Download ZIP."

**[Скриншот: кнопка Download ZIP на GitHub]**

---

### Часть 3: Создание виртуального окружения (2:00 - 2:45)

**[Действие: открыть терминал в папке проекта]**

**Диктор:**
> "Рекомендую создать виртуальное окружение, чтобы изолировать зависимости проекта."

**[Действие: ввод команд]**

```bash
# Для Windows:
python -m venv venv
venv\Scripts\activate

# Для macOS/Linux:
python3 -m venv venv
source venv/bin/activate
```

**Диктор:**
> "Вы увидите (venv) в начале строки — это значит, что окружение активировано."

**[Скриншот: терминал с (venv)]**

---

### Часть 4: Установка зависимостей (2:45 - 3:30)

**[Действие: ввод команды]**

```bash
pip install -r requirements.txt
```

**Диктор:**
> "Эта команда установит все необходимые библиотеки. Это может занять несколько минут..."

**[Ускоренная запись процесса установки]**

**Диктор:**
> "Отлично, все зависимости установлены!"

---

### Часть 5: Настройка API-ключа (3:30 - 4:30)

**[Действие: открыть браузер, перейти на the-odds-api.com]**

**Диктор:**
> "Теперь нам нужен API-ключ для получения реальных коэффициентов. Перейдите на the-odds-api.com и зарегистрируйтесь."

**[Действие: показать процесс регистрации]**

**Диктор:**
> "Бесплатный план даёт 500 запросов в месяц — этого достаточно для начала."

**[Действие: скопировать API-ключ]**

**Диктор:**
> "Скопируйте ваш API-ключ из личного кабинета."

**[Действие: открыть config/config.yaml]**

```yaml
api:
  the_odds_api:
    key: "ВАШ_API_КЛЮЧ_ЗДЕСЬ"
```

**Диктор:**
> "Вставьте ключ в файл конфигурации и сохраните его."

---

### Часть 6: Первый запуск (4:30 - 5:30)

**[Действие: запустить приложение]**

```bash
python main_gui.py
```

**Диктор:**
> "Запускаем графический интерфейс..."

**[Скриншот: главное окно приложения]**

**Диктор:**
> "И вот мы видим главное окно Eden Analytics Pro! Установка завершена."

---

### Заключение (5:30 - 6:00)

**[Скриншот: обзор интерфейса]**

**Диктор:**
