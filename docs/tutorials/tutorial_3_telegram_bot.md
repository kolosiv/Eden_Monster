# 🎥 Видео-туториал #3: Настройка Telegram-бота

## 🎬 Метаданные

- **Длительность:** 6-8 минут
- **Уровень:** Начинающий
- **Предыдущее видео:** Первый арбитраж
- **Формат:** Запись экрана + голос за кадром

---

## 📑 Сценарий

### Введение (0:00 - 0:30)

**[Заставка с логотипом]**

**Диктор:**
> "Привет! В этом видео мы настроим Telegram-бота, который будет отправлять вам уведомления о новых арбитражных возможностях прямо в Telegram."

**[Скриншот: уведомление в Telegram]**

---

### Часть 1: Создание бота (0:30 - 2:30)

**[Действие: открыть Telegram]**

**Диктор:**
> "Для создания бота нам понадобится BotFather — это официальный бот Telegram для управления ботами."

**[Действие: поиск @BotFather]**

**Диктор:**
> "В поиске Telegram набираем '@BotFather'. Важно выбрать именно официального бота с синей галочкой."

**[Скриншот: BotFather в поиске]**

**[Действие: отправить /newbot]**

**Диктор:**
> "Нажимаем Start и отправляем команду /newbot для создания нового бота."

**[Скриншот: ответ BotFather]**

**Диктор:**
> "BotFather просит ввести имя бота. Введём 'Eden Analytics Bot'."

**[Действие: ввести имя]**

**Диктор:**
> "Теперь нужно выбрать уникальный username. Он должен заканчиваться на 'bot'. Например, 'eden_analytics_bot'."

**[Действие: ввести username]**

**[Скриншот: успешное создание с токеном]**

**Диктор:**
> "Отлично! Бот создан. Самое важное — это токен. Скопируйте его — он понадобится для настройки."

**[Действие: скопировать токен]**

---

### Часть 2: Получение User ID (2:30 - 3:30)

**[Действие: поиск @userinfobot]**

**Диктор:**
> "Теперь нам нужен ваш Telegram User ID. Найдите бота '@userinfobot'."

**[Действие: отправить /start]**

**[Скриншот: ответ с User ID]**

**Диктор:**
> "Бот показывает ваш ID — это числовой идентификатор. Скопируйте его."

---

### Часть 3: Настройка конфигурации (3:30 - 5:00)

**[Действие: открыть config/config.yaml]**

**Диктор:**
> "Откроем файл конфигурации config/config.yaml и найдём секцию telegram."

**[Скриншот: секция telegram в config.yaml]**

**Диктор:**
> "Вот что нужно изменить:
> 1. `enabled` — меняем на `true`
> 2. `bot_token` — вставляем токен от BotFather
> 3. `admin_ids` — вставляем ваш User ID"

**[Действие: редактирование файла]**

```yaml
telegram:
  enabled: true
  bot_token: "123456789:ABCdefGhIjKlmNoPqRsTuVwXyZ"
  admin_ids:
    - 123456789
```

**Диктор:**
> "Сохраняем файл."

---

### Часть 4: Запуск бота (5:00 - 6:00)

**[Действие: открыть терминал]**

**Диктор:**
> "Теперь запустим бота. Есть два способа."

**[Действие: ввести команду]**

```bash
python start_bot.py
```

**[Скриншот: сообщение об успешном запуске]**

**Диктор:**
> "Бот запущен! Видим сообщение 'Bot started successfully'."

---

### Часть 5: Тестирование (6:00 - 7:30)

**[Действие: открыть Telegram, найти бота]**

**Диктор:**
> "Найдём нашего бота по username и нажмём Start."

**[Действие: отправить /start]**

**[Скриншот: приветственное сообщение]**

**Диктор:**
> "Бот отвечает! Давайте попробуем найти арбитраж."

**[Действие: отправить /arbitrage]**

**[Скриншот: результаты поиска в Telegram]**

**Диктор:**
> "Бот нашёл возможности и прислал их прямо в чат! Теперь вы можете получать уведомления на телефон."

---

### Заключение (7:30 - 8:00)

**[Скриншот: список команд]**

**Диктор:**
> "Основные команды бота:
> - /start — начало работы
> - /arbitrage — поиск возможностей
> - /stats — ваша статистика
> - /settings — настройки уведомлений"

**[Заставка]**

**Диктор:**
> "На этом всё! Теперь вы полностью готовы к работе с Eden Analytics Pro. Удачи!"
