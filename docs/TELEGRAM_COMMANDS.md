# 📱 Команды Telegram-бота Eden Analytics

> Полный список команд для работы с Telegram-ботом

---

## 👤 Пользовательские команды

### Основные

| Команда | Описание |
|---------|----------|
| `/start` | 👋 Начало работы с ботом |
| `/help` | ❓ Справка по всем командам |
| `/register` | 📝 Регистрация нового пользователя |
| `/profile` | 👤 Просмотр профиля |

### Арбитраж

| Команда | Описание |
|---------|----------|
| `/arbitrage` | 🔍 Найти арбитражные возможности |
| `/arbitrage <roi>` | 🔍 Найти с мин. ROI |
| `/today` | 📅 Матчи сегодня |
| `/live` | 🟢 Текущие возможности |

**Примеры:**
```
/arbitrage        # Найти все возможности
/arbitrage 3      # Найти с ROI ≥ 3%
```

### Статистика

| Команда | Описание |
|---------|----------|
| `/stats` | 📊 Общая статистика |
| `/stats today` | 📊 Статистика за сегодня |
| `/stats week` | 📊 Статистика за неделю |
| `/stats month` | 📊 Статистика за месяц |
| `/history` | 📋 История ставок |
| `/history <n>` | 📋 Последние N ставок |

**Примеры:**
```
/stats           # Общая статистика
/stats week      # За неделю
/history 10      # Последние 10 ставок
```

### Банкролл

| Команда | Описание |
|---------|----------|
| `/bankroll` | 💰 Текущий банкролл |
| `/bankroll <сумма>` | 💰 Установить банкролл |
| `/deposit <сумма>` | ➕ Пополнить банкролл |
| `/withdraw <сумма>` | ➖ Снять с банкролла |

**Примеры:**
```
/bankroll         # Показать текущий банкролл
/bankroll 1500    # Установить $1500
/deposit 500      # Пополнить на $500
/withdraw 200     # Снять $200
```

### Настройки

| Команда | Описание |
|---------|----------|
| `/settings` | ⚙️ Меню настроек |
| `/notify on` | 🔔 Включить уведомления |
| `/notify off` | 🔕 Выключить уведомления |
| `/roi <значение>` | 🎯 Установить мин. ROI |
| `/hole <значение>` | ⚠️ Установить макс. hole risk |

**Примеры:**
```
/roi 2.5          # Мин. ROI = 2.5%
/hole 4           # Макс. hole risk = 4%
```

### Подписка

| Команда | Описание |
|---------|----------|
| `/subscribe` | ⭐ Информация о подписках |
| `/premium` | 💎 Премиум-функции |

## 👑 Администраторские команды

> ℹ️ Доступны только для администраторов (ID в admin_ids)

### Управление

| Команда | Описание |
|---------|----------|
| `/admin` | 👑 Панель администратора |
| `/users` | 👥 Список пользователей |
| `/users active` | 👥 Активные пользователи |
| `/user <id>` | 👤 Инфо о пользователе |
| `/stats_global` | 📊 Глобальная статистика |

### Модерация

| Команда | Описание |
|---------|----------|
| `/ban <user_id>` | ⛔ Заблокировать пользователя |
| `/unban <user_id>` | ✅ Разблокировать |
| `/premium <user_id>` | 💎 Выдать премиум |
| `/revoke <user_id>` | ❌ Отозвать премиум |

**Примеры:**
```
/ban 123456789        # Заблокировать пользователя
/unban 123456789      # Разблокировать
/premium 123456789    # Выдать премиум
```

### Рассылка

| Команда | Описание |
|---------|----------|
| `/broadcast` | 📢 Рассылка всем пользователям |
| `/broadcast_premium` | 📢 Рассылка премиум-пользователям |
| `/message <id> <текст>` | ✉️ Сообщение пользователю |

**Пример:**
```
/broadcast
🔔 Обновление системы!

Добавлены новые функции:
• Улучшенные ML-прогнозы
• Поддержка KHL
```

### Система

| Команда | Описание |
|---------|----------|
| `/logs` | 📝 Последние логи |
| `/logs <n>` | 📝 Последние N записей |
| `/status` | 🟢 Статус системы |
| `/restart` | 🔄 Перезапустить бота |
| `/config` | ⚙️ Текущая конфигурация |
