# ⌨️ Горячие клавиши Eden Analytics Pro

> Справочник горячих клавиш для быстрой работы с приложением

---

## 📝 Основные команды

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+F` | 🔍 Найти арбитраж | Запустить поиск арбитражных возможностей |
| `Ctrl+R` | 🔄 Обновить | Обновить данные с API |
| `F5` | 🔃 Обновить страницу | Перезагрузить текущую страницу |
| `Ctrl+Q` | 🚪 Выход | Закрыть приложение |
| `Escape` | ❌ Закрыть | Закрыть диалог/всплывающее окно |

---

## 📁 Файлы и экспорт

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+E` | 📤 Экспорт | Экспорт данных (CSV/Excel/PDF) |
| `Ctrl+S` | 💾 Сохранить | Сохранить настройки |
| `Ctrl+O` | 📂 Открыть | Открыть файл |
| `Ctrl+P` | 🖨️ Печать | Печать отчёта |

---

## 📊 Навигация

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+1` | 🏠 Главная | Перейти на дашборд |
| `Ctrl+2` | 🔍 Поиск | Перейти к поиску арбитража |
| `Ctrl+3` | 📊 История | Перейти к истории ставок |
| `Ctrl+4` | 📈 Статистика | Перейти к статистике |
| `Ctrl+5` | 🤖 ML-модели | Перейти к ML-моделям |
| `Ctrl+6` | 💰 Банкролл | Перейти к управлению банкроллом |
| `Ctrl+,` | ⚙️ Настройки | Открыть настройки |

---

## 🗑️ Таблицы

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `↑` / `↓` | Навигация | Перемещение по строкам |
| `Enter` | Открыть | Открыть детали выбранной строки |
| `Delete` | Удалить | Удалить выбранную запись |
| `Ctrl+A` | Выбрать всё | Выделить все строки |
| `Ctrl+C` | Копировать | Копировать выделенное |
| `Home` | В начало | Перейти в начало таблицы |
| `End` | В конец | Перейти в конец таблицы |

---

## 📅 Даты и фильтры

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+D` | Сегодня | Показать данные за сегодня |
| `Ctrl+W` | Неделя | Показать данные за неделю |
| `Ctrl+M` | Месяц | Показать данные за месяц |
| `Ctrl+Shift+F` | Фильтры | Открыть панель фильтров |

---

## 🎙️ Бэктестинг

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+B` | Запуск | Запустить бэктест |
| `Ctrl+Shift+B` | Остановка | Остановить бэктест |

---

## 💡 Справка

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `F1` | Справка | Открыть документацию |
| `Ctrl+H` | История | Показать историю |
| `Ctrl+?` | Сокращения | Показать этот список |
