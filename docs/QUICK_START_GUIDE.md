## 🔧 Метод 1: Установка из исходников (Рекомендуется для разработчиков)

### Шаг 1: Загрузка проекта

**Вариант A: Клонирование через Git**
```bash
git clone https://github.com/username/eden-analytics-pro.git
cd eden-analytics-pro
```

**Вариант B: Скачивание архива**
1. Скачайте ZIP-архив проекта
2. Распакуйте в удобную директорию
3. Откройте терминал/командную строку в этой директории

### Шаг 2: Создание виртуального окружения (рекомендуется)

**Windows:**
```cmd
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### Шаг 3: Установка зависимостей

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

> 💡 **Совет:** Если возникают ошибки при установке PyQt6, попробуйте:
> ```bash
> pip install PyQt6 --only-binary :all:
> ```

### Шаг 4: Настройка конфигурации

1. Откройте файл `config/config.yaml`
2. Введите ваш API-ключ от The Odds API:

```yaml
api:
  the_odds_api:
    key: "ВАШ_API_КЛЮЧ_ЗДЕСЬ"
```

> 🔑 **Получение API-ключа:**
> 1. Перейдите на https://the-odds-api.com/
> 2. Зарегистрируйтесь (бесплатный план: 500 запросов/месяц)
> 3. Скопируйте ваш API-ключ

### Шаг 5: Инициализация базы данных

```bash
python -c "from database.db_manager import DatabaseManager; DatabaseManager()"
```

### Шаг 6: Запуск приложения

**Графический интерфейс (GUI):**
```bash
python main_gui.py
```

**Консольный интерфейс:**
```bash
python main.py
```

### Шаг 7: Обучение ML-модели (опционально)

```bash
python -c "from models.model_trainer import train_initial_model; train_initial_model()"
```

## 📦 Метод 2: Установщик для Windows (Рекомендуется для конечных пользователей)

### Создание установщика (для разработчиков)

#### Шаг 1: Установка PyInstaller

```bash
pip install pyinstaller
```

#### Шаг 2: Сборка исполняемого файла

```bash
python installer/windows/setup.py
```

#### Шаг 3: Установка Inno Setup

1. Скачайте Inno Setup с https://jrsoftware.org/isdl.php
2. Установите программу
3. Добавьте путь к ISCC.exe в переменную PATH

#### Шаг 4: Создание установщика

```bash
build_installer.bat
```

#### Результат

После успешной сборки:
```
Output/EdenAnalyticsPro_Setup_v2.0.0.exe
```

### Установка (для конечных пользователей)

#### Шаг 1: Скачивание

Скачайте `EdenAnalyticsPro_Setup_v2.0.0.exe` из раздела Releases.

#### Шаг 2: Запуск установщика

1. Дважды щёлкните по файлу установщика
2. Следуйте инструкциям мастера установки:

```

│     Мастер установки Eden Analytics Pro    │
├─────────────────────────────────────────────┤
│                                             │
│  [●] Лицензионное соглашение               │

│  [ ] Создание ярлыков                      │
```

3. Выберите папку установки (по умолчанию: `C:\Program Files\Eden Analytics Pro`)

## 🔑 Настройка API-ключей

### The Odds API (обязательно для реальных данных)

1. Перейдите на https://the-odds-api.com/
2. Нажмите **Get Started**
3. Выберите план:
   - **Free** (бесплатно): 500 запросов/месяц
   - **Starter** ($19/мес): 5,000 запросов
   - **Pro** ($79/мес): 25,000 запросов

4. Зарегистрируйтесь и подтвердите email
5. Скопируйте API-ключ из личного кабинета

### NHL API (опционально, для статистики)

NHL API бесплатен и не требует ключа:
```
https://statsapi.web.nhl.com/api/v1/
```

### Конфигурация

Отредактируйте `config/config.yaml`:

```yaml
# Демо-режим (без API-ключа)
demo_mode: false  # Установите false для реальных данных

# API конфигурация
api:
  the_odds_api:
    key: "ваш-api-ключ-здесь"
    base_url: "https://api.the-odds-api.com/v4"
    sport: "icehockey_nhl"
    regions: "us,eu"
```

## ❌ Устранение неполадок

### Ошибка: "ModuleNotFoundError: No module named 'PyQt6'"

**Решение:**
```bash
pip uninstall PyQt6 PyQt6-Qt6 PyQt6-sip
pip install PyQt6
```

### Ошибка: "API key is invalid"

**Решение:**
1. Проверьте правильность ключа в `config/config.yaml`
2. Убедитесь, что нет лишних пробелов
3. Проверьте, не истёк ли ключ на сайте The Odds API

### Ошибка: "Database is locked"

**Решение:**
```bash
# Удалите файл блокировки
rm -f eden_mvp.db-journal

# Или пересоздайте базу данных
python -c "from database.db_manager import DatabaseManager; DatabaseManager()"
```

### Приложение не запускается на macOS

**Решение:**
```bash
# Разрешите запуск в системных настройках
xattr -cr /Applications/Eden\ Analytics\ Pro.app
```

### Telegram-бот не отвечает

**Решение:**
1. Проверьте токен бота в конфигурации
2. Убедитесь, что `enabled: true` в секции telegram
3. См. раздел [TELEGRAM_BOT_SETUP.md](TELEGRAM_BOT_SETUP.md)
