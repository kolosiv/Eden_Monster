## 🔧 Шаг 1: Создание бота в Telegram

### 1.1 Открытие BotFather

1. Откройте Telegram на телефоне или компьютере
2. В поиске введите: `@BotFather`
3. Нажмите на официального бота (с синей галочкой ✓)
4. Нажмите **Start** или отправьте `/start`

```
┌─────────────────────────────────────────────────────────────────────┐
│  @BotFather                                              [✓]      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  🤖 BotFather                                                      │
│                                                                     │
│  I can help you create and manage Telegram bots.                   │
│                                                                     │
│  /newbot - create a new bot                                        │
│  /mybots - manage your bots                                        │
│  /setname - change bot's name                                      │
│  /setdescription - change bot description                          │
│  /setabouttext - change about info                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.2 Создание нового бота

1. Отправьте команду: `/newbot`

2. **Введите имя бота** (отображаемое имя):
   ```
   Eden Analytics Bot
   ```
   
3. **Введите username бота** (уникальный, должен заканчиваться на `bot`):
   ```
   eden_analytics_bot
   ```
   
   > ⚠️ Если username занят, попробуйте другой вариант:
   > - `eden_arb_bot`
   > - `my_eden_bot`
   > - `eden_hockey_bot`

4. **Получите токен бота**:

```
┌─────────────────────────────────────────────────────────────────────┐
│  @BotFather                                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Done! Congratulations on your new bot. You will find it at        │
│  t.me/eden_analytics_bot.                                          │
│                                                                     │
│  Use this token to access the HTTP API:                            │
│                                                                     │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ 123456789:ABCdefGhIjKlmNoPqRsTuVwXyZ-1234567890           │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  Keep your token secure and store it safely.                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

5. **Скопируйте токен** — он понадобится для настройки

### 1.3 Настройка описания бота (опционально)

Отправьте BotFather следующие команды:

```
/setdescription
```
Выберите бота и введите описание:
```
🏒 Eden Analytics Pro - система анализа хоккейного арбитража

📊 Автоматический поиск арбитражных возможностей
🤖 ML-прогнозы овертаймов
💰 Умное управление банкроллом
📱 Уведомления в реальном времени
```

```
/setabouttext
```
Введите краткое описание:
```
Умный анализ хоккейных ставок с ML-прогнозами
```

## ⚙️ Шаг 2: Настройка конфигурации

### 2.1 Получение вашего Telegram User ID

1. Откройте Telegram
2. Найдите бота `@userinfobot` или `@getmyid_bot`
3. Отправьте `/start`
4. Скопируйте ваш User ID (числовой)

```
┌─────────────────────────────────────────────────────────────────────┐
│  @userinfobot                                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Your user info:                                                    │
│                                                                     │
│  ID: 123456789                                                      │
│  First name: Иван                                                  │
│  Last name: Иванов                                                 │
│  Username: @ivan_ivanov                                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.2 Редактирование конфигурации

Откройте файл `config/config.yaml` и найдите секцию `telegram`:

```yaml
# Telegram Bot Settings
telegram:
  enabled: true  # ← Измените на true
  bot_token: "YOUR_BOT_TOKEN_HERE"  # ← Вставьте ваш токен
  admin_ids:
    - 123456789  # ← Вставьте ваш User ID
    - 987654321  # ← Дополнительные админы (опционально)
  
  # Настройки уведомлений
  notifications:
    new_arbitrage: true     # Уведомления о новых возможностях
    match_results: true     # Результаты матчей
    daily_summary: true     # Ежедневный отчёт
    risk_alerts: true       # Предупреждения о рисках
  
  # Расписание уведомлений
  schedule:
    daily_summary_time: "09:00"  # Время ежедневного отчёта
    timezone: "Europe/Moscow"
  
  # Ограничения
  rate_limit:
    messages_per_minute: 20
    arbitrage_min_interval: 300  # 5 минут между уведомлениями
```

### 2.3 Пример готовой конфигурации

```yaml
telegram:
  enabled: true
  bot_token: "123456789:ABCdefGhIjKlmNoPqRsTuVwXyZ-1234567890"
  admin_ids:
    - 123456789

    new_arbitrage: true
    match_results: true
    daily_summary: true
    risk_alerts: true
  
  schedule:
    daily_summary_time: "09:00"
    timezone: "Europe/Moscow"
```

## ❌ Устранение неполадок

### Проблема: "Unauthorized" при запуске бота

**Причина:** Неверный токен бота

**Решение:**
1. Проверьте токен в `config/config.yaml`
2. Убедитесь, что нет лишних пробелов
3. Получите новый токен у @BotFather: `/revoke`

### Проблема: Бот не отвечает на сообщения

**Проверьте:**
1. Бот запущен? (`ps aux | grep multi_user_bot`)
2. `enabled: true` в конфигурации?
3. Нет ли ошибок в логах?

**Просмотр логов:**
```bash
tail -f logs/eden.log
```

### Проблема: "Chat not found"

**Причина:** User ID не в списке admin_ids

**Решение:**
1. Получите ваш ID через @userinfobot
2. Добавьте его в `config/config.yaml`:
```yaml
telegram:
  admin_ids:
    - 123456789  # ваш ID
```

### Проблема: Дублирование сообщений

**Причина:** Запущено несколько экземпляров бота

**Решение:**
```bash
# Остановите все экземпляры
pkill -f multi_user_bot

# Запустите один экземпляр
python telegram_bot/multi_user_bot.py
```

### Проблема: Бот медленно отвечает

**Возможные причины:**
1. Медленное интернет-соединение
2. Перегрузка API The Odds
3. Много активных пользователей

**Решение:**
```yaml
# Увеличьте лимиты в config.yaml
telegram:
  rate_limit:
    messages_per_minute: 30
    arbitrage_min_interval: 180
```
