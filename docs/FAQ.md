## 🌐 Общие вопросы

### Что такое Eden Analytics Pro?

Eden Analytics Pro — это система анализа хоккейного арбитража, которая:
- Находит арбитражные возможности в NHL
- Прогнозирует вероятность овертайма с помощью ML
- Оценивает риски ("дыра" / hole)
- Рассчитывает оптимальные ставки

### Что такое "дыра" (hole)?

"Дыра" — это сценарий, при котором обе ставки проигрывают:
1. Вы ставите на победу фаворита (включая ОТ)
2. Вы ставите на победу аутсайдера в основное время
3. Если ничья в основное время, а аутсайдер выигрывает в ОТ — это "дыра"

**Схема:**
```
Основное время: Ничья (2:2)
Овертайм: Аутсайдер выигрывает
Результат: Обе ставки проигрывают → "HOLE"
```

### Нужен ли опыт в ставках?

Нет, система автоматически:
- Находит возможности
- Оценивает риски
- Рассчитывает суммы ставок
- Даёт чёткие рекомендации

### Это гарантированный доход?

Нет. Арбитраж в хоккее не является безрисковым из-за вероятности "дыры". 
Однако при правильном управлении рисками система даёт положительное математическое ожидание.

## 💻 Установка и настройка

### Как получить API-ключ?

1. Перейдите на https://the-odds-api.com/
2. Зарегистрируйтесь (бесплатно)
3. Подтвердите email
4. Скопируйте ключ из личного кабинета

Бесплатный план: 500 запросов в месяц

### Можно использовать без API-ключа?

Да, в демо-режиме. В `config/config.yaml`:

```yaml
demo_mode: true
```

В демо-режиме используются тестовые данные.

### Почему PyQt6 не устанавливается?

Попробуйте:

```bash
pip install PyQt6 --only-binary :all:
```

Или установите системные зависимости (Linux):

```bash
sudo apt-get install python3-pyqt6
```

### Как обновить приложение?

```bash
git pull origin main
pip install -r requirements.txt --upgrade
```

## 🏒 Арбитраж и ставки

### Какой минимальный ROI рекомендуется?

Рекомендуемый минимум: **2%**

При ROI ниже 2% прибыль может не покрыть риск "дыры".

### Какой максимальный hole risk допустим?

Рекомендуемый максимум: **4%**

При hole risk > 4% риск слишком высок.

### Почему некоторые возможности помечены "ОСТОРОЖНО"?

Это значит:
- Hole risk близок к порогу (3-4%)
- Или уверенность модели ниже 70%
- Или есть дополнительные факторы риска

### Как часто обновлять данные?

Рекомендуется:
- Перед размещением ставки — всегда
- Каждые 15-30 минут в игровые дни
- Коэффициенты меняются быстро!

### Как выбрать стратегию ставок?

| Стратегия | Описание | Когда использовать |
|-----------|----------|-------------------|
| Арбитражная | Равный выигрыш при любом исходе | По умолчанию |
| Kelly | Максимизация роста банкролла | Для опытных |
| Фиксированный % | Одинаковый % от банкролла | Для начинающих |

## 🤖 ML-модели

### Зачем нужна ML-модель?

ML-модель улучшает прогнозирование вероятности овертайма, учитывая 16+ факторов:
- Статистику команд
- Историю овертаймов
- Усталость и форму
- Специальные бригады
- И другое

### Как обучить модель?

```bash
python -c "from models.model_trainer import train_initial_model; train_initial_model()"
```

Обучение занимает 1-2 минуты.

### Что если ML-модель не загружена?

Система автоматически использует запасную модель Пуассона.
Она менее точна, но работает без дополнительной настройки.

### Как часто переобучать модель?

Рекомендуется:
- Каждые 100 новых матчей
- Или раз в неделю
- Или при падении точности на 5%+

## 📱 Telegram-бот

### Как создать бота?

1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

Подробнее: [TELEGRAM_BOT_SETUP.md](TELEGRAM_BOT_SETUP.md)

### Как узнать свой Telegram ID?

1. Найдите бота @userinfobot
2. Отправьте `/start`
3. Скопируйте ваш ID

### Бот не отвечает, что делать?

1. Проверьте, запущен ли бот: `ps aux | grep multi_user_bot`
2. Проверьте `enabled: true` в конфигурации
3. Проверьте логи: `tail -f logs/eden.log`

## 🛠️ Решение проблем

### "База данных заблокирована"

```bash
# Удалите файл блокировки
rm -f eden_mvp.db-journal
```

1. Проверьте ключ на https://the-odds-api.com/
2. Убедитесь, что нет лишних пробелов
3. Проверьте лимит запросов

### Приложение не запускается

Запустите диагностику:

```bash
python check_system.py
```

### Как получить логи?

```bash
# Последние 100 строк
tail -n 100 logs/eden.log

# В реальном времени
tail -f logs/eden.log
```

## 💡 Советы и лучшие практики

### Начинающим

1. ✅ Начните с демо-режима
2. ✅ Используйте небольшой банкролл для тестов
3. ✅ Следуйте рекомендациям системы
4. ❌ Не игнорируйте предупреждения "ОСТОРОЖНО"
5. ❌ Не ставьте больше 10% банкролла на одну сделку

1. 🎯 Настройте ML-модель
2. 📊 Используйте бэктестинг для проверки стратегий
3. 📊 Анализируйте статистику регулярно
4. 🔄 Переобучайте модель еженедельно
5. 📤 Экспортируйте данные для анализа

1. 💰 Установите дневной лимит потерь
2. 📉 Снижайте ставки при просадке
3. 🚫 Не гонитесь за потерями
4. 📅 Ведите историю всех ставок
5. 🎯 Следуйте стратегии, а не эмоциям
