## ⚠️ Выявленные проблемы (из PDF анализа)

### 🔴 Критические проблемы (ИСПРАВЛЕНО)

| № | Проблема | Статус | Решение |
|---|----------|--------|---------|
| 1 | **AUC = 1.0 на обучающей выборке** - признак data leakage | ✅ Исправлено | Удалены blacklisted features (predicted_closeness, implied_closeness) |
| 2 | **OT Rate = 0% в тестовой выборке** - некорректные данные | ✅ Исправлено | Добавлена валидация OT rate (15-35%) для всех датасетов |
| 3 | **Маржа букмекера не учтена в EV** | ✅ Исправлено | Добавлено вычитание маржи 6.5% из расчёта EV |
| 4 | **SMOTE на всём датасете** | ✅ Исправлено | SMOTE применяется только внутри CV-фолдов |
| 5 | **Scaler fit на всём датасете** | ✅ Исправлено | Scaler обучается только на train-части каждого фолда |
| 6 | **Playoff игры в обучении** | ✅ Исправлено | Playoff фильтруется (другие правила OT) |
| 7 | **Данные до 2015** | ✅ Исправлено | Данные до 2015 фильтруются (нет 3-on-3 OT) |

### 🟠 Серьёзные проблемы (ИСПРАВЛЕНО)

| № | Проблема | Статус | Решение |
|---|----------|--------|---------|
| 1 | Только 5 trials Optuna | ✅ Исправлено | Увеличено до 100 trials |
| 2 | Monte Carlo 1000 симуляций | ✅ Исправлено | Увеличено до 10,000+ |

```python

- Playoff игры (непрерывный OT, нет shootout)
- Игры до сезона 2015-2016 (нет 3-on-3 OT)
- Игры с OT rate < 15% или > 35% (аномалия)
```

### 5. Валидация OT Rate

```python
def validate_ot_rate(ot_count, total_games, dataset_name):
    ot_rate = ot_count / total_games
    MIN_EXPECTED = 0.15  # 15%
    MAX_EXPECTED = 0.35  # 35%
    
    if ot_rate < MIN_EXPECTED:
        return False, "OT rate слишком низкий - возможна ошибка данных"
    if ot_rate > MAX_EXPECTED:
        return False, "OT rate слишком высокий - возможны playoff игры"
    return True, "OK"
```

## 🎯 Рекомендации по использованию

### Безопасное использование модели

1. **Размер ставки**: 1-3% от банкролла максимум
2. **Stop-loss**: Прекратить ставки при просадке 15%
3. **Верификация**: Всегда проверять коэффициенты перед ставкой
4. **Диверсификация**: Не ставить весь банкролл на одну стратегию

### Ограничения модели

```
⚠️ ВАЖНЫЕ ОГРАНИЧЕНИЯ:
- Модель предсказывает вероятности, не гарантии
- Исторические результаты не гарантируют будущих
- Лимиты букмекеров могут ограничить реальные ставки
- Рыночные условия могут измениться
- Модель оптимизирована для регулярного сезона NHL
```
